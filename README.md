# propymepostevento
Link del proyecto: https://ptapialavin.github.io/propymepostevento/ 

📌 PROPYME Landing Page – Workshop 2026
Proyecto desarrollado como parte del módulo CSS Avanzado del curso Desafío Latam – Desarrollo Full Stack JavaScript, correspondiente a la prueba final del módulo.
🧩 Descripción
Este proyecto consiste en una landing page informativa para PROPYME (www.propyme.com), diseñada para centralizar y compartir contenido relevante con los asistentes de encuentros presenciales.
La página reúne recursos clave del 4° Workshop PROPYME 2026, incluyendo:

Presentaciones oficiales
Nota de prensa
Galería de imágenes del evento
Video de la actividad
Enlaces a sitio web y redes sociales

🎯 Objetivo
Crear una interfaz moderna, responsiva y visualmente atractiva utilizando CSS avanzado, enfocada en mejorar la experiencia del usuario al acceder a información posterior al evento.
🛠️ Tecnologías utilizadas

HTML5 semántico
CSS3 (Flexbox, Grid, variables CSS, media queries)
Font Awesome (iconografía)

💡 Características principales

✅ Layout con CSS Grid para estructura general
✅ Diseño responsivo adaptable a distintos dispositivos
✅ Galería de imágenes interactiva con efectos hover (escala + filtro)
✅ Menú lateral (sidebar) en versión desktop
✅ Uso de variables CSS para consistencia de estilos
✅ Transiciones suaves para mejorar la experiencia visual

📱 Comportamiento responsivo


Mobile (<576px)

Layout vertical
Menú tipo hamburguesa (actualmente sólo visual)



Desktop (≥576px)

Layout de dos columnas
Sidebar de navegación visible
Galería optimizada (4 columnas)



⚠️ Nota importante
El menú hamburguesa en versión móvil no despliega el menú de navegación, debido a que se requiere implementación con JavaScript (función toggleMenu() no incluida en este proyecto).
